# SiteGlobalSolution2025.2
Desenvolvimento de uma landing page para uma empresa de contratação. Projeto com o íntuito de praticar a utilização de conceitos do desenvolvimento WEB, como: responsividade, darkMode, consumo de JSON e Node. Nele, temos uma plataforma simples de contratação, onde são disponibilizados dados referentes a 60 possíveis funcionários, onde o usuário pode filtrar dados desse usuário de acordo com sua preferência.

# Técnologias utilizadas:
- React
- Node
- TailwindCSS
- Vite
- Git/Github
- Vercel

# Instalação

## Como Iniciamos o processo: 
#### Só afim de curiosidade, para sua execução não se deve seguir esses passos.

### Criação do projeto
```bash
npm create vite@latest "Nome do Projeto"
```

### Localiazção do diretório
```bash
cd "Nome do projeto"
```

### Instalação Node Package Manager
```bash
npm i
```

### Instalação do Tailwind
```bash
npm install tailwindcss @tailwindcss/vite
```
ps: realizar importação no vite.config

### Instalação React-icons
```bash
npm i react-icons
```

### Execução do projeto
```bash
npm run dev
```

## Rodar em sua Máquina Local:
Descompactar o arquivo -> Abrir em seu editor de código -> digitar no terminal: "cd lokidin" -> digitar no terminal "npm i" -> 
rodar o projeto "npm run dev"
